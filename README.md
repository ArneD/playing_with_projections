# Playing with Projections
Preparation of a workshop

## Server
The implementation server is a sinatra app.

To prepare the server:
```
cd server
bundle install
```

To run the server:
```
ruby server/app.rb
```

## Ruby client
To prepare the client:
```
cd client/ruby
bundle install
```

To run the client
```
ruby clients/ruby/client.rb
```
